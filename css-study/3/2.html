蓝色的部分宽度为100px 红色部分宽度不定, 为自适应宽度. 可以随着屏幕的大小变化而变化

方法一

.one{ background-color: blue; width: 100px; position: absolute; top: 0;bottom: 0; }

.two{ background-color: red; position: absolute; top: 0; bottom: 0; left: 100px; right: 0px; }

方法二

.in{ background-color: blue; width: 100px; position: absolute; top: 0;bottom: 0; }

.out{ background-color: red; width: 100%; height: 100%; }

.left{ background-color: blue; width: 100px; height: 400px; float:left; }

.right{ margin: 0 0 0 100px; }

.right_in{ background-color: red; float: left; height: 400px; width:100%; }
